Homework 5¶

I've chosen Alcohol Effects On Study dataset from Kaggle competition. For this particular homework I've used only Maths.csv dataframe. My prediction model of choice is XGBRegressor which is a tree-based ensemble regression model. Model's task is to predict diffrence between final grade and second year grade marked as "G3" based on other observations. Target values spans from 0 to 20.

Important note in this task I'm performing my predictions differently then in previous assignments. I try to predict difference between G3 and G2 with removed information about G1 and G2 substituded by info about their difference.

(1)¶

The most important variables according to PVI are:

  • Absences => Attendance is very important part of learning since if we skip classes we'll be behind,
  • Age => The older you get, you're more likely to repeat a class since people rarely change,
  • Famrel => Having issues with your family is a huge factor in shifting the focus of young people away from studying,
  • Activities_yes => Doing activities outside of school like sport or additional classes gives you more advantage over your colleagues,
  • failures => Failing a class or an exam can motivate you to work harder or make you don't care about school at all if number of fails is high,
  • G2 (difference) => If we improved from class 1 to 2 than we're likely to improve from class 2 to 3.

Let's compare these results to different models.

(2)¶

To show pvi for different models I've prepared 3 more examples:

  • Using set from previous task => predict G3 knowing G2 and G1

As we can see here G2 is a dominant variable here since the best prediction is just predict the same grade as G2. Next three features are absences, age and activities_yes which are variables that have 1st, 2nd and 4th importance in its PVI.

  • Changing hyperparameters by adding more depth to XGBRegressor

Its results are very similar to model in (1), but G2 has lower importance and Medu variable is marked as more important, which is mother education. Mothers are more likely to help their children studying while the father is working so this importance makes sense.

  • Changing model to AdaBoostRegressor

Here we see some changes to the most important variables. Activities feature got more importance score than femrel, but this diffrence is really small. This model also values more than G2 (increase in grades between year 2 and 1) features like: amount of freetime, health and mothers education.
