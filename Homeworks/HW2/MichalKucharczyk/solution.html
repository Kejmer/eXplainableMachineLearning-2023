Task B¶

Pasted value functions:
v() = 0
v(A) = 20
v(B) = 20
v(C) = 60
v(A,B) = 60
v(A,C) = 70
v(B,C) = 70
v(A,B,C) = 100

There is only 1 way to extend a 0-sized set by A and it brings 20 there
A can extend 1-element sets in two ways here and on average it brings (60 - 20 + 70 - 60) / 2 = 25
A can extend 2-element set in only one way putting additional 100 - 70 = 30

So finally SHAP value for A is the average of the above, so (20 + 25 + 30) / 3 = 25
