Task 1¶

Let Y be a RV that a person will use XAI, Y_hat - RV our prediction if will use XAI (if a person is enrolled in training for blues). Let C be the group (either blue or red).

demografic parity coeff = P(Y_hat = 1 | C = blue) / P(Y_hat = 1 | C = red) = 0.65 / 0.5 = 1.3

equal opportunity coeff = P(Y_hat = 1 | Y = 1, C = blue) / P(Y_hat = 1 | Y = 1, C = red) = 0.75 / 0.5 = 1.5

predictive rate parity coeff = P(Y = 1 | Y_hat = 0, C = blue) / P(Y = 1 | Y_hat = 1, C = red) = 0.92 / 0.5 = 1.84

I would say that the easiest thing to do would be to increase the percentage of Red people allowed to attend XAI training. The increase from 50% to 75% (still randomly selected) would let all three disparity coefficients lie in [4/5, 5/4] interval.
